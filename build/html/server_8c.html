Detailed Description

Simple TCP/IP server that deals with multiple clients. Simulate an robocomm server.

Based on http://www.linuxhowtos.org/C_C++/socket.htm A simple server in the internet domain using TCP The port number is passed as an argument This version runs forever, forking off a separate process for each connection
